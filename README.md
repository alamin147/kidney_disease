This project focuses on predicting Chronic Kidney Disease (CKD) using machine learning techniques. The dataset was preprocessed by handling missing values, encoding categorical variables, and normalizing features to improve model performance.

A baseline model was first developed, achieving an accuracy of 95.00%. To further enhance performance, data balancing using SMOTE (Synthetic Minority Oversampling Technique) and multiple feature selection techniques were applied. Methods such as Chi-Square and Recursive Feature Elimination (RFE) improved the model accuracy to 98.33% and 99.17%, respectively.

Additionally, hyperparameter tuning was performed, resulting in a stable accuracy of 99.00%. To ensure the model’s robustness and generalization capability, cross-validation techniques were used. The model achieved 98.29% ± 2.10 accuracy with 5-fold cross-validation and 98.00% ± 2.23 with 10-fold cross-validation.

Overall, the project highlights the importance of data preprocessing, feature selection, and model optimization in building reliable healthcare prediction systems.
